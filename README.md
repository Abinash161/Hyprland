### To use this Hyprland Rice

```sh
git clone https://github.com/Abinash161/Hyprland.git &&
cd Hyprland	&&
rm -rf ~/.config &&
cp -r .config ~/.config	&&
cd .local  &&
cp -r bin ~/.local/bin &&
hyprctl reload
```
